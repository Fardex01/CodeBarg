ساختار صفحات HTML

هر فایل HTML با این ساختار کلی شروع می‌شود:

<!DOCTYPE html>
<html lang="fa">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>کدبرگ</title>
  </head>
  <body>
    <h1>کدبرگ</h1>
  </body>
</html>
